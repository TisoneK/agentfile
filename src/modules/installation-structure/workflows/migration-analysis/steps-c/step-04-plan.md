# Step 4: Create Migration Plan

## STEP GOAL

Generate a detailed step-by-step migration strategy.

## MANDATORY EXECUTION RULES

1. **Sequence steps** - Order migration activities
2. **Define procedures** - Document how to perform each step
3. **Set milestones** - Identify key checkpoints

## EXECUTION PROTOCOLS

### 1. Sequence Steps

Order migration activities:
- Pre-migration tasks
- Core migration steps
- Post-migration tasks
- Validation steps

### 2. Define Procedures

For each step, document:
- What to do
- How to do it
- Expected outcome
- Rollback procedure

### 3. Set Milestones

Define checkpoints:
- Pre-migration complete
- Core migration complete
- Post-migration complete
- Validation passed

## OUTPUT

Migration plan document with step-by-step procedures.
