# Step 2: Migration Preparation

## STEP GOAL

Set up the target .agentfile/ directory structure for migration.

## MANDATORY EXECUTION RULES

1. **Create structure** - Set up new directory layout
2. **Prepare configs** - Configure for migration
3. **Ready for migration** - Ensure target is prepared

## EXECUTION PROTOCOLS

### 1. Create Structure

Create directories:
- `.agentfile/` root
- `.agentfile/agents/`
- `.agentfile/workflows/`
- `.agentfile/skills/`
- `.agentfile/config/`
- `.agentfile/memory/`

### 2. Prepare Configs

Set up:
- Module configuration
- Agent configurations
- Workflow configurations

### 3. Ready for Migration

Confirm:
- Structure created
- Configs in place
- Ready to receive files

## OUTPUT

Target structure ready.

## MENU OPTIONS

**[C] Continue** - Proceed to file migration
