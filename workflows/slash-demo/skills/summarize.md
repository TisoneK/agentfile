# Skill: Summarize

## Purpose
Teach agents how to create effective, concise summaries from input topics.

## Instructions
1. **Analyze the Input**: Read and understand the topic thoroughly
2. **Identify Key Points**: Extract 3-5 most important concepts
3. **Structure the Summary**: Use the agent's specified format
4. **Be Concise**: Keep total summary under 150 words
5. **Maintain Accuracy**: Only include information from the input

## Examples

**Input**: "Artificial Intelligence"
**Output**:
```markdown
# Summary: Artificial Intelligence

## Key Points
- AI refers to computer systems that can perform tasks requiring human intelligence
- Machine learning is a subset of AI that enables systems to learn from data
- AI applications include natural language processing, computer vision, and robotics

## Brief Overview
Artificial Intelligence encompasses technologies that enable machines to simulate human intelligence and learning capabilities.
```

**Input**: "Climate Change"
**Output**:
```markdown
# Summary: Climate Change

## Key Points
- Climate change refers to long-term shifts in global temperatures and weather patterns
- Human activities, particularly fossil fuel burning, are the primary cause
- Impacts include rising sea levels, extreme weather, and ecosystem disruption

## Brief Overview
Climate change represents significant alterations in Earth's climate system, primarily driven by human-induced greenhouse gas emissions.
```
