# Skill: Analyze

## Purpose
Teach agents how to perform effective topic analysis with structured output.

## Instructions
1. **Understand the Topic**: Read and comprehend the input thoroughly
2. **Identify Key Concepts**: Extract 3-5 main concepts
3. **Find Insights**: Look for relationships and patterns
4. **Structure Output**: Use the agent's specified format
5. **Be Concise**: Keep analysis focused and clear

## Examples

**Input**: "Artificial Intelligence"
**Output**:
```markdown
# Analysis: Artificial Intelligence

## Key Concepts
- Machine Learning algorithms
- Neural networks and deep learning
- Natural language processing
- Computer vision applications

## Insights
AI encompasses technologies that enable machines to simulate human intelligence through learning and adaptation.

## Summary
Artificial Intelligence represents the development of computer systems that can perform tasks requiring human intelligence.
```

**Input**: "Climate Change"
**Output**:
```markdown
# Analysis: Climate Change

## Key Concepts
- Greenhouse gas emissions
- Global temperature rise
- Renewable energy transition
- Climate adaptation strategies

## Insights
Climate change represents long-term shifts in global weather patterns driven primarily by human activities.

## Summary
Climate change refers to significant alterations in Earth's climate system caused by increased greenhouse gas concentrations.
```
