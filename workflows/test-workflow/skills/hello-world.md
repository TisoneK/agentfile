---
name: Hello World Skill
description: Generates a friendly hello world greeting
---

## Task
Generate a friendly hello world greeting based on the user's input.

## Input
User's natural language request (available as $USER_REQUEST)

## Output
A warm greeting message saved to outputs/greeting.md

## Steps
1. Read the user's request
2. Generate a friendly hello world greeting
3. Personalize it based on the request context
4. Save the greeting to the output file

## Example Output
```
Hello World! Welcome to your Agentfile test workflow. 

I see you're interested in: [user's request topic]

Let's get started with this exciting journey!
```
