# Analysis Skill

## Purpose
Perform comprehensive multi-perspective analysis of any given topic.

## Process
1. **Topic Decomposition**: Break down the topic into fundamental components
2. **Technical Analysis**: Examine technical aspects, requirements, and considerations
3. **Practical Assessment**: Evaluate real-world applications and implementation possibilities
4. **Challenge Identification**: Identify potential obstacles, risks, and limitations
5. **Synthesis**: Combine all perspectives into a cohesive analysis

## Output Structure
- **Overview**: Brief introduction to the topic
- **Technical Perspective**: Technical details and considerations
- **Practical Applications**: Real-world uses and implementations
- **Challenges & Risks**: Potential obstacles and mitigation strategies
- **Conclusions**: Key findings and implications

## Guidelines
- Be thorough but concise
- Provide specific examples where relevant
- Consider both benefits and drawbacks
- Focus on actionable insights
