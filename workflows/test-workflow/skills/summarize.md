# Summarization Skill

## Purpose
Create concise, actionable summaries from detailed analysis content.

## Process
1. **Content Review**: Thoroughly review the source analysis
2. **Key Point Extraction**: Identify the most important insights and findings
3. **Prioritization**: Rank information by importance and actionability
4. **Synthesis**: Combine key points into a coherent summary
5. **Refinement**: Ensure clarity, conciseness, and impact

## Output Structure
- **Executive Summary**: 2-3 sentence overview
- **Key Points**: 3-5 bullet points of critical information
- **Actionable Insights**: Specific recommendations or next steps
- **Impact Assessment**: Brief note on significance and implications

## Guidelines
- Keep it under 150 words total
- Use clear, direct language
- Focus on what matters most
- Ensure every point adds value
- Make it immediately useful for decision-making
