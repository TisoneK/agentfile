# Step 5: Validation

## STEP GOAL

Verify the migration was successful and everything works correctly.

## MANDATORY EXECUTION RULES

1. **Verify structure** - Check new directory layout
2. **Test functionality** - Ensure agents/workflows work
3. **Generate report** - Document validation results

## EXECUTION PROTOCOLS

### 1. Verify Structure

Check:
- All directories present
- Files in correct locations
- Configuration files valid

### 2. Test Functionality

Test:
- Agent loading
- Workflow execution
- Configuration applied

### 3. Generate Report

Create:
- Structure verification
- Functionality tests
- Issues found (if any)

## OUTPUT

Validation report.

## MENU OPTIONS

**[C] Continue** - Proceed to cleanup (if validation passes)
