# Step 4: Configuration Update

## STEP GOAL

Update configurations to work with the new structure.

## MANDATORY EXECUTION RULES

1. **Update paths** - Fix file paths in configs
2. **Migrate settings** - Transfer user preferences
3. **Apply new settings** - Add new configuration options

## EXECUTION PROTOCOLS

### 1. Update Paths

Fix references to:
- Agent file locations
- Workflow paths
- Skill directories

### 2. Migrate Settings

Transfer:
- User preferences
- Custom configurations
- Integration settings

### 3. Apply New Settings

Add:
- New validation settings
- Installation preferences
- Module configurations

## OUTPUT

Configurations updated for new structure.

## MENU OPTIONS

**[C] Continue** - Proceed to validation
