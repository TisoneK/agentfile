# Step 4: Validation

## STEP GOAL

Test template generation and installation process.

## MANDATORY EXECUTION RULES

1. **Test generation** - Does template generate correctly?
2. **Test installation** - Does agentfile install work?
3. **Verify structure** - Are all files in place?

## EXECUTION PROTOCOLS

### 1. Test Generation

Generate a test project:
- Run template with defaults
- Check file structure
- Verify configuration

### 2. Test Installation

Test agentfile setup:
- Run clean installation
- Verify .agentfile/ created
- Check configuration applied

### 3. Verify Structure

Confirm:
- All required files present
- Correct directory layout
- Proper file permissions

## OUTPUT

Validation report with test results.

**[C] Continue** - Proceed to package template
