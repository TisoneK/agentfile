# Step 3: Customization Options

## STEP GOAL

Configure template variables and user-customizable settings.

## MANDATORY EXECUTION RULES

1. **Define variables** - What can users customize?
2. **Set defaults** - Provide sensible defaults
3. **Document options** - Explain customization in docs

## EXECUTION PROTOCOLS

### 1. Define Variables

Determine customizable elements:
- Project name
- Author/maintainer
- Framework version
- Agentfile preferences

### 2. Set Defaults

Configure sensible defaults:
- Installation options
- Validation settings
- Agent configurations

### 3. Document Options

Create documentation:
- How to customize
- Available variables
- Example configurations

## OUTPUT

Customization configuration and documentation.
