# Step 1: Template Configuration

## STEP GOAL

Define the project template structure and configuration options.

## MANDATORY EXECUTION RULES

1. **Identify template type** - What kind of project?
2. **Define structure** - What folders/files to include?
3. **Set options** - What can users customize?

## EXECUTION PROTOCOLS

### 1. Identify Template Type

Ask user:
- Project framework/language (Node.js, Python, etc.)
- Template purpose (API, web app, library, etc.)
- Target use case

### 2. Define Structure

Determine:
- Required directories
- Essential files
- Configuration templates

### 3. Set Options

Configure:
- Template variables
- Customizable settings
- Default values

## OUTPUT

Template configuration document.
