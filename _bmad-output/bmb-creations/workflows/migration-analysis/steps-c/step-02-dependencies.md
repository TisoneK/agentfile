# Step 2: Analyze Dependencies

## STEP GOAL

Identify tools, workflows, and customizations that need to be migrated.

## MANDATORY EXECUTION RULES

1. **Identify tools** - What MCP tools are in use?
2. **Map workflows** - Which workflows are active?
3. **Catalog customizations** - What local modifications exist?

## EXECUTION PROTOCOLS

### 1. Identify Tools

From scan results, identify:
- MCP tools configured
- External integrations
- Custom scripts

### 2. Map Workflows

Determine:
- Active workflow definitions
- Workflow configurations
- Scheduled/running workflows

### 3. Catalog Customizations

Document:
- Modified agent files
- Custom prompts
- Local configurations

## OUTPUT

Dependency inventory with migration requirements.

## MENU OPTIONS

**[C] Continue** - Proceed to assess migration complexity
